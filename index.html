Password Tester

Use this page to check if the password you want to use is unique.
This site uses the Have I Been Pwned API which contains over 501 Million passwords obtained through various data breaches. If the password you enter is found, it means that somewhere a hacker has it as well.

Nerd Talk

This site does not send the passwords you enter anywhere, it uses the first 5 characters of SHA-1 hash of the password, utilizing the k-anonymity aspect of the HIBP API. Once a list of hashes is returned from the API, we locally compare the hashes to see if your password is in the list
